# 洪荒长生 - 李长寿传

一款以太白金星李长寿为主角的洪荒修仙游戏Demo。

## 游戏特色

- 🗺️ **无缝大世界** - 4000x4000像素的开放世界地图
- 🎮 **高自由度** - 自由探索洪荒世界
- ⚔️ **战斗系统** - 技能释放、经验升级
- 💬 **NPC对话** - 与玉鼎真人、云霄师姐、杨戬等角色互动
- 📱 **跨平台** - 支持浏览器和iOS设备

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动 |
| Shift | 冲刺 |
| E | 交互/对话 |
| 1/2/3 | 使用技能 |
| ESC | 打开菜单 |
| 点击敌人 | 进入战斗 |

## 游戏世界观

### 区域
- **终南山·清虚观** - 李长寿修行的山门
- **东海之滨** - 碧波万顷，仙岛林立
- **昆仑山** - 万山之祖，仙气浓郁
- **火云洞** - 三皇居所，威严神圣

### 修炼境界
炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 炼虚期 → 合体期 → 大乘期 → 渡劫期 → 真仙 → 金仙 → 太乙金仙 → 大罗金仙 → 准圣 → 圣人

## 技能系统

- 🍃 **长生大道** - 核心功法，可长生不老
- ☯️ **太极图** - 先天至宝，演化阴阳
- ⚔️ **青萍剑** - 先天灵宝，杀伐无双
- 🗡️ **真武大帝法身** - 召唤法身助战

## 开发环境

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建项目
npm run build
```

## iOS打包

```bash
# 初始化Cordova项目
npm run cordova:init

# 构建iOS应用
npm run cordova:build
```
